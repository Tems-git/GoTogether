# GoTogether

Семейни пътувания без главоболие.

## Стек

- React Native + Expo SDK 54
- Supabase (PostgreSQL + Auth + Storage)
- Claude API (Anthropic)

## База данни — 6 таблици с RLS

profiles, trips, trip_members, expenses, expense_splits, documents
